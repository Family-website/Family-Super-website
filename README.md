# 🏠 GharManager

🚀 **The Ultimate Household Financial & Data Management App**

GharManager is a comprehensive, offline-ready Progressive Web App (PWA) designed to simplify and digitize daily household tracking. From managing monthly budgets to calculating complex local interests, this app acts as a secure, cloud-synced digital ledger for the entire family.

## ✨ Key Features
* **💰 Smart Hisaab:** Track daily expenses with category-wise sorting, member tags, and real-time monthly budget alerts.
* **📊 Visual Reports:** Interactive charts (Doughnut & Pie) to analyze spending habits, with 1-click professional PDF exports.
* **🥛 Daily Dudh Record:** Log morning and evening milk intake with automatic monthly billing and rate calculation.
* **🛒 Ration List (Smart Sync):** Manage grocery lists with an auto-sync feature that adds purchased items directly to the main expense ledger.
* **🏦 Financial Calculators:** Built-in tools for Bank EMI and local Village Interest (Rupay Saikra).
* **☁️ Real-time Cloud Sync:** Powered by Google Firebase Authentication and Firestore for secure, real-time cross-device data backup.
* **🎙️ Voice Typing:** Built-in Web Speech API integration to quickly add expense details without typing.
* **📱 Native PWA Experience:** Installable directly to the home screen with dynamic themes, sound effects, and a secure lock screen.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom UI/UX), Vanilla JavaScript
* **Backend & Database:** Google Firebase (Auth & Firestore)
* **Libraries:** Chart.js, jsPDF, SweetAlert2, Canvas-Confetti
